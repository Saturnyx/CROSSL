# CROSSL Licenses

A collection of software and program licenses used for RECF events. These
licenses are consistent with the RECF Student-Centered Policy and the VEX EULA,
and they aim to allow competition-legal software while discouraging misuse.

# License List

| **License** | CROSSL-P-1.0 | CROSSL-WC-1.0
| :--- | :--- | :--- |
| **License Type** | Permissive | Weak Copyleft |
| **Visibility of Derivative Works** | Can be open-source or closed-source | Only the Program & derivative works should be open-source |
| **Viral Nature** | No | No |
| **Commercial Use** | Allowed | Allowed |

# General Summary

Most of the license follows the terms and conditions of the MIT License. Key
points:

- Permissive: use/copy/modify/merge/publish/distribute/sublicense/sell allowed
  (subject to terms)
- Attribution required: include copyright + license text in copies/substantial
  portions
- Warranty/LIability: provided “AS IS”; no warranty; authors not liable
- Scope: license covers only the Program; VEX hardware/software/services remain
  under VEX EULA/TOU
- Robotics context:
  - Distinguishes **Robotics-Specific Code** vs **Generic Code**
  - In RECF competition context, Students must be able to
    explain/modify/reimplement Robotics-Specific Code
  - Adults must not undermine student learning (e.g., writing Competition Code
    for students, in-competition unfair assistance, or making code
    inaccessible/opaque)
- Termination:
  - Strict immediate termination (no cure) for deliberate Adult violation re:
    authoring Competition Code for a registered student team
  - Otherwise typically 30-day cure; technical issues handled via notice + cure
- Non-endorsement: don’t use authors/contributors/team names/numbers to promote
  without written permission
- Versioning: steward-controlled official CROSSL versions; projects can pin a
  version or use “or later”; declared incompatibilities may restrict mixing
  versions/variants

# Current Situation and Future Plans

Currently, VEX Robotics and the RECF are going through a split that may affect
the terms of the license (especially since this license was based on the RECF
student sentered policy). However, not much is known about this situation as of
now. Therefore, the current licenses may continued to be used for the
foreseeable future, but they may need to be updated depending on how the
situation unfolds. Visit this forum thread for more discussion on this topic:
https://www.vexforum.com/t/rec-foundation-no-longer-runs-vex-events/146301
