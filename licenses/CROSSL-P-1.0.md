## 6. Educational Use and Student-Centered Policy

This section of the license specifically addresses the use of the Program in
the context of the RECF's student-centered policy, which emphasizes the
importance of student learning and development in competitive robotics. The
license encourages the use of the Program in a manner that supports and enhances
the educational experience of students.

### 6.1. Use of the Program by Students

Students may use the Program for learning, development, and
participation in competitive robotics events. Where the Program
is used in an RECF-sanctioned competition context, Students
must be able to:

 - display their capability to understand and explain the Robotics-Specific Code
   they are using, rather than relying solely on the Program as a "black box"
   solution.
 - demonstrate their ability to modify and adapt the Robotics-Specific Code to
   suit their specific robot design and competition strategy if needed.
 - demonstrate their ability to rewrite or reimplement the Robotics-Specific
   Code on their own if necessary, to show that they have a comprehensive
   understanding of the code and its functionality.

(Generic Code may be used without these requirements, as it is not specific to
robotics and does not directly impact the educational experience in the same
way.)

### 6.2. Use of the Program by Adults

Adults shall use the Program in a manner that supports and enhances the
educational experience of students. Adults shall not use the Program in a way
that undermines the learning and development of students. Prohibited uses
include, but are not limited to:

 - Using the Program to create Competition Code for students without their
   involvement or understanding.
 - Using the Program to provide direct assistance to students during
   competitions in a way that violates competition rules or gives an unfair
   advantage.
 - Authoring Competition Code in a manner that is not accessible or
   understandable to students, thereby hindering their learning and development.
